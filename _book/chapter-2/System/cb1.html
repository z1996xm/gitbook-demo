CB1

一、BIGTREETECH CB1 Procedure

1.1 Download system image

If you use the CB1 core board, you can only download and install the system image we provide:

https://github.com/bigtreetech/CB1/releases

1.2 Download and install Raspberry Pi Imager

Download and install the official burning software https://www.raspberrypi.com/software/ of Raspberry Pi System of CB1

Images can also be burned using this software.

1.3 Burning system

  1. Insert the Micro SD card reader into the computer.

  2. Select system

  1. Select "User defined", and then select the image downloaded to the computer

  1. Select the Micro SD card to be burned (burning the image will format the Micro SD card. Be careful not to select the wrong disk

Otherwise the data on other storage will be formatted), click "Burn"
